# 訂餐外送系統

## 系統利害關係人（System stakeholders）

| 類別 | 利害關係人 | 關注重點 | 對系統的核心需求 |
|---|---|---|---|
| 核心使用者 | 顧客 | 訂餐便利性、餐點多樣化、付款安全性、即時追蹤、售後服務 | 能快速搜尋餐廳與菜單；支援多種付款方式（信用卡、電子支付）；提供訂單狀態追蹤（外送進度）；可聯繫客服處理訂單問題 |
| 核心使用者 | 餐廳 | 訂單資訊完整、操作介面簡潔、營收管理、顧客評價 | 接單與備餐狀態更新介面；菜單與價格管理功能；訂單明細與銷售報表；顧客評價檢視與回覆 |
| 核心使用者 | 外送員 | 派單公平性、路線導航準確、收益透明、即時溝通 | 即時派單通知與路線導航；外送進度更新功能；外送完成回報與收益查詢；聯繫顧客與客服的通訊介面 |
| 系統所有者與管理者 | 平台營運方 | 平台營運績效、用戶增長、詐欺防範、系統穩定性 | 取得營運數據與報表分析；促銷活動與優惠券管理；使用者與餐廳帳號管理；平台舞弊偵測與稽核系統 |
| 系統所有者與管理者 | 客服 | 處理訂單糾紛、退款申請、即時支援 | 查詢訂單詳情與交易記錄；處理退款、補償與申訴；內建即時溝通工具（與顧客/餐廳/外送員） |
| 系統所有者與管理者 | 技術開發團隊 | 系統穩定、可維護性、可擴展性、監控與除錯 | 系統日誌與監控模組；測試與部署環境；微服務架構與 API 文件；故障警示與版本管理 |
| 外部相關單位 | 支付服務提供商 | 交易安全性、串接穩定性、法規遵循 | 安全 API 串接與錯誤回傳機制；交易驗證與對帳流程；符合 PCI-DSS 標準 |
| 外部相關單位 | 政府與監管機關 | 消費者保護、個資法遵循、食品安全、稅務合規 | 提供報表稅務查詢介面；個資加密與權限控管；食品業者與配送資料紀錄 |
| 外部相關單位 | 投資人／股東 | 平台成長率、用戶留存率、營收與獲利 | 平台財務報表匯出；系統績效指標儀表板（KPI Dashboard） |
